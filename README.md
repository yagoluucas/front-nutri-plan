# NutriPlan

Este é um projeto para gerenciamento e envios de dietas.

## Bibliotecas utilizadas no Front-end

- **Next.js**: framework principal do front-end.
- **Tailwind CSS**: estilização da interface.
- **daisyUI**: componentes prontos baseados em Tailwind.
- **react-hook-form**: controle de formulários.
- **zod**: validação dos dados dos formulários.
- **@hookform/resolvers**: integração entre react-hook-form e zod.
- **@tanstack/react-query**: chamadas para API, cache e atualização de dados.
- **lucide-react**: ícones da interface.
- **sonner**: notificações/toasts de sucesso e erro.
- **@react-pdf/renderer**: geração e download do plano alimentar em PDF.
